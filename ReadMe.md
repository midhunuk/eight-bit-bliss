# Eight Bit Bliss 🎮

Eight Bit Bliss is a Nintendo Entertainment System (NES) emulator written in Rust.  
This project is built as a learning exercise to explore systems programming, computer architecture, and emulator design.

It follows the excellent guide *NES Emulator from Scratch* by Michael Fogleman (Bugzmanov):  
https://bugzmanov.github.io/nes_ebook/

---

## 🚀 Motivation

This project started as a way to learn Rust while exploring how classic game consoles work internally.

Building a NES emulator is a hands-on way to understand:
- Low-level CPU design
- Memory management
- Bitwise operations
- System architecture
- Performance considerations in Rust

And of course — it’s also satisfying to see something gradually behave like a real console.

---

## 🧠 Current Progress

The project is in its very early stages.

- The CPU emulation has been started  
- The first opcode has been implemented  

More instructions and system components are actively being developed.

## 📌 Goals

- Implement a working NES CPU (6502)
- Add PPU (graphics) emulation
- Run basic NES ROMs
- Improve accuracy and performance over time
- Strengthen Rust and systems programming skills

This project is currently in early development.
